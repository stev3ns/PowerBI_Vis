# Power BI portfoolio

See repositoorium koondab minu Power BI projekte. Dashboardid ja raportilehed on koostatud eesti keeles ning eesmärk on näidata Power BI oskusi (visualiseerimine, andmemudel, DAX, Power Query) praktiliste näidete kaudu.

## Projektid

### 1) Ilmajaamad
Kursuse käigus valminud projekt, mis on koostatud juhendi järgi.

- Kaust: `Ilmajaamad/`
- Failid: `.pbix` (Power BI raport) + pildid (eelvaated)

### 2) Juhtimislaud (prototüüp)
Prototüüp-juhtimislaud, mis on tehtud **genereeritud (fake) andmetega**.  
Raporti sisu (KPI-d, vaated ja diagrammide valik) on üles ehitatud **päris tootmisettevõtte müügiprotsessi kitsaskohtade** loogika järgi — eesmärk on demonstreerida, kuidas probleemidest lähtudes kujundada toimiv juhtimisvaade.

- Kaust: `Juhtimislaud/`
- Failid: `.pbix` + pildid (eelvaated)

## Kuidas avada
1. Ava projekti kaust.
2. Laadi alla `.pbix` fail.
3. Ava fail **Power BI Desktopis**.
4. Vajadusel uuenda andmeallika ühendused `Power Query` alt.
